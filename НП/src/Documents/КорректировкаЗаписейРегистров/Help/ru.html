Корректировка регистров

Документ предназначен для ручной корректировки записей регистров накопления и регистров сведений.

Список документов открывается Настройки и администрирование – Первоначальное заполнение ИБ – Корректировка регистров. С помощью сервиса Найти, который запускается из командной панели списка, Пользователь может выбрать нужные документы, задавая поиск по значениям различных реквизитов.

Новый документ можно добавить с помощью кнопки Создать в командной панели списка.

По кнопке "Настройка состава регистров" верхней командной панели формы открывается форма настройки состава регистров. Флажками отмечаются те регистры, записи по которым планируется ввести.

На форме документа появляются закладки, соответствующие выбранным регистрам. В табличные поля на закладках вводятся необходимые данные в соответствии со структурой данных, предусмотренных в регистрах.

При проведении документа корректируется информация в тех регистрах, которые указаны в документе. С помощью кнопки "Движения документа" в панели навигации формы можно проконтролировать те движения, которые произведены после проведения документа.

Важно. Документ является служебным документом и доступен для заполнения только пользователю, который обладает полными правами (администратору системы).
